# Closet Organization Guide 

## What is this?
 The Closet Organization Guide is a straightforward, interactive web application designed to help users categorize their clothing effectively. The site allows users to filter different types of outfits, such as work, going out, workout, dresses, and winter wear. It provides a visual and structured way to view and manage clothing categories in one place.

## Why does it exist? 
This project was created to solve the common problem of closet clutter and decision fatigue when choosing outfits. By organizing clothing into clear categories, users can quickly find what they need and streamline their daily routine. The system is also based on my personal closet organization method, making it practical and easy to follow.
